# Analizator japoński tytułów anime

## Uruchomienie projektu:
1. Zainstawanie zależności: `pip install -r requirements.txt`
2. Uruchomienie serwera: `uvicorn main:app --reload`
3. Aplikacja dostępna pod adresem: `http://127.0.0.1:8000`

## Doumentacja (OpenAPI)
Dokumentacja generowania automatycznie (Swagger), dostępna pod adresem  `http://127.0.0.1:8000/docs`
